# Exercício 02 — Objeto (instância)

## Objetivo

Entender o que é um **objeto** e o papel da palavra `new` na criação de uma instância.

## Arquivos

- `Aluno.java`: a classe (molde).
- `Principal.java`: cria um objeto com `new` e imprime mensagens.

## O que fazer

- Leia o enunciado em `Exercicio_02_Objeto.md`.
- Rode o `Principal.java`.
- No código, identifique:
  - tipo: `Aluno`
  - variável: `objetoAluno`
  - criação do objeto: `new Aluno()`
- Responda no `Resposta_Exercicio_02.md` (ou no seu caderno) as perguntas do enunciado.

## Como executar (Windows)

Na raiz do projeto:

```bat
javac Exercicio02_Objeto\Aluno.java Exercicio02_Objeto\Principal.java
java Exercicio02_Objeto.Principal
```
