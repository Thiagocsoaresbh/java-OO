# Exercício 03 — A palavra `new` e múltiplos objetos

## Objetivo

Perceber que `new` é o que **cria o objeto na memória** e que é possível criar **vários objetos** do mesmo tipo.

## Arquivos

- `Aluno.java`: atributos `nome` e `idade`.
- `Principal.java`: cria `a1` e `a2` com `new` e imprime os dois.

## O que fazer

- Leia o enunciado em `Exercicio_03_New.md`.
- Rode o `Principal.java`.
- Compare a saída de `a1` e `a2` e entenda que são **dois objetos diferentes**.
- Responda no `Resposta_Exercicio_03.md` (ou no seu caderno) as perguntas do enunciado.

## Como executar (Windows)

Na raiz do projeto:

```bat
javac Exercicio03_New\Aluno.java Exercicio03_New\Principal.java
java Exercicio03_New.Principal
```
