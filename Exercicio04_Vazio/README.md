# Exercício 04 — Objeto “vazio” e valores padrão

## Objetivo

Entender que, ao nascer, um objeto tem **valores padrão** nos atributos:

- `String` → `null`
- `int` → `0`

## Arquivos

- `Aluno.java`
- `Principal.java`: cria um `Aluno` e imprime os atributos sem preencher nada.

## O que fazer

- Leia o enunciado em `Exercicio_04_Vazio.md`.
- Rode o `Principal.java`.
- Observe os valores impressos.
- Responda no `Resposta_Exercicio_04.md` (ou no seu caderno) as perguntas do enunciado.

## Como executar (Windows)

Na raiz do projeto:

```bat
javac Exercicio04_Vazio\Aluno.java Exercicio04_Vazio\Principal.java
java Exercicio04_Vazio.Principal
```
